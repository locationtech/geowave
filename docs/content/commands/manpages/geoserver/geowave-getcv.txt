NAME
//:----

geowave gs cv get - Get a GeoServer coverage's info

SYNOPSIS
//:--------

geowave gs cv get [options] <coverage name>

DESCRIPTION
//:-----------

This command will return a GeoServer coverage's info

OPTIONS
//:-------

- -cs, --coverageStore
 * <coverage store name>
- -ws, --workspace
 * <workspace name>
