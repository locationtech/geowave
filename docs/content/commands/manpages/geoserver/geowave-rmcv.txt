NAME
//:----

geowave gs cv rm - Remove a GeoServer coverage

SYNOPSIS
//:--------

geowave gs cv rm [options] <coverage name>

DESCRIPTION
//:-----------

This command will remove a GeoServer coverage

OPTIONS
//:-------

- * -cs, --cvgstore
 * <coverage store name>
- -ws, --workspace
 * <workspace name>
