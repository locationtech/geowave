NAME
//:----

geowave gs ds rm - Remove GeoServer DataStore

SYNOPSIS
//:--------

geowave gs ds rm [options] <datastore name>

DESCRIPTION
//:-----------

This command will remove a GeoServer DataStore

OPTIONS
//:-------

- -ws, --workspace
 * <workspace name>
