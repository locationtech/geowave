NAME

geowave gs cs list - List GeoServer coverage stores

SYNOPSIS
//:--------

geowave gs cs list [options]

DESCRIPTION
//:-----------

This command will list all GeoServer coverage stores

OPTIONS
//:-------

- -ws, --workspace
 * <workspace name>
