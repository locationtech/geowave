NAME
//:----

geowave gs listcv - List GeoServer Coverages

SYNOPSIS
//:--------

geowave gs listcv [options] <coverage store name>

DESCRIPTION
//:-----------

This command will list all GeoServer Coverages

OPTIONS
//:-------

- -ws, --workspace
 * <workspace name>
