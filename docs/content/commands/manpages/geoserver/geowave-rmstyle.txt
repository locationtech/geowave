NAME
//:----

geowave gs style rm - Remove GeoServer Style

SYNOPSIS
//:--------

geowave gs style rm [options] <style name>

DESCRIPTION
//:-----------

This command will remove a GeoServer Style

OPTIONS
//:-------

There are currently no options available for this command
