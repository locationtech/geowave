NAME
//:----

geowave gs ds list - List GeoServer datastores

SYNOPSIS
//:--------

geowave gs ds list [options]

DESCRIPTION
//:-----------

This command will list all GeoServer datastores

OPTIONS
//:-------

- -ws, --workspace
 * <workspace name>
