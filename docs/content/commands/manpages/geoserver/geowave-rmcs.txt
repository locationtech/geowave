NAME
//:----

geowave gs rmcs - Remove GeoServer Coverage Store

SYNOPSIS
//:--------

geowave gs rmcs [options] <coverage store name>

DESCRIPTION
//:-----------

This command will remove a GeoServer Coverage Store

OPTIONS
//:-------

- -ws, --workspace
 * Workspace Name
