NAME
//:----

geowave gs ds get - Get GeoServer DataStore info

SYNOPSIS
//:--------

geowave gs ds get [options] <datastore name>

DESCRIPTION
//:-----------

This command will return a GeoServer DataStore info

OPTIONS
//:-------

- -ws, --workspace
 * <workspace name>
