NAME
//:----

geowave config addindexgrp - Group multiple index configurations together given a name

SYNOPSIS
//:--------

geowave config addindexgrp [options] <name> <comma separated list of indexes>

DESCRIPTION
//:-----------

The geowave config addindexgrp operator will group multiple index configurations together given a name.
This acts as a convenience for re-using multiple indices together on ingest.

OPTIONS
//:-------

There are currently no options for this command
