    public abstract class GeoWaveJobRunner
extends org.apache.hadoop.conf.Configured
implements org.apache.hadoop.util.Tool
    This class can run a basic job to query GeoWave. It manages Accumulo user, Accumulo password, Accumulo instance name, zookeeper URLs, Accumulo namespace, adapters, indices, query, min splits and max splits.
