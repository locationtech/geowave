Class SimpleLongIndexStrategy

  • All Implemented Interfaces:
    IndexStrategy<MultiDimensionalNumericData,MultiDimensionalNumericData>, NumericIndexStrategy, Persistable

    public class SimpleLongIndexStrategy
extends SimpleNumericIndexStrategy<Long>
    A simple 1-dimensional NumericIndexStrategy that represents an index of signed long values. The strategy doesn't use any binning. The ids are simply the byte arrays of the value. This index strategy will not perform well for inserting ranges because there will be too much replication of data.
